# Hydra (Android)

## Quickstart
* Clone the repo, i.e. `git clone git@github.com:ZeusWPI/hydra-android.git`
* Download Android Studio
* Choose 'Import Project (gradle etc.)'
* Choose the local folder
* Have fun :)
